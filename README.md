# How to Run this website on your local machine

Clone this repo into your machine and navigate to the folder of this project and run npm install to install required modules to run the project then open your chrome browser and visit this link - https://chromewebstore.google.com/detail/allow-cors-access-control/lhobafahddgcelffkeicbaginigeejlf 
download the extension as it is necessary to the run the project. after downloading enable the extension.
then do npm run start on your terminal inside project folder.
now go on your browser and search for localhost:1234 you will see the project.
